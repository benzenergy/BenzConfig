# BenzConfig

Приложение для расчёта расхода топлива (лето / зима) на основе
пробега, пропорций движения и норм расхода.

## Возможности
- Раздельный расчёт для лета и зимы
- Настраиваемые пропорции город / трасса
- Сохранение настроек
- Анимация вывода результата

## Платформа
- Windows 10 / 11
- .NET 8 (WPF)

## Установка
Скачайте готовый файл из раздела **Releases** и запустите.
